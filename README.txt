DULCES AROMAS - POS COLIBRI v4.3
=====================================

INSTRUCCIONES PARA REINALDO
----------------------------

1. DESCARGAR Y DESCOMPRIMIR
   - Descarga el archivo colibri-pos-v4.3.zip
   - Descomprime en tu computadora
   - Verás una carpeta "colibri-pos-v4.3" con archivos dentro

2. SUBIR A GITHUB PAGES
   - Entra a github.com e inicia sesión
   - Crea un nuevo repositorio (botón verde "New")
   - Nombre: dulces-aromas-pos (o el que prefieras)
   - Público
   - NO agregar README ni .gitignore
   - Click "Create repository"
   - En la página siguiente, busca "uploading an existing file"
   - Arrastra TODOS los archivos de la carpeta descomprimida
   - Click "Commit changes"
   - Ve a Settings (pestaña arriba)
   - En el menú izquierdo busca "Pages"
   - En "Source" selecciona "Deploy from a branch"
   - Selecciona "main" y carpeta "/ (root)"
   - Click "Save"
   - Espera 2-5 minutos
   - Tu link aparecerá arriba: https://tusuario.github.io/dulces-aromas-pos/

3. USAR EN EL CELULAR
   - Abre Chrome (Android) o Safari (iPhone)
   - Ve al link de GitHub Pages
   - Agrega a pantalla de inicio:
     * Android: Menú (3 puntos) → "Agregar a pantalla de inicio"
     * iPhone: Compartir → "Agregar a pantalla de inicio"
   - ¡Listo! Funciona como app nativa

4. BACKUP DIARIO (IMPORTANTE)
   - Ve a Configuración (último botón abajo)
   - Click "Exportar Backup"
   - Se descarga un archivo .json
   - Guárdalo en Google Drive o envíalo por WhatsApp
   - Si cambias de celular o borras datos, usa "Importar Backup"

5. DATOS DE PRUEBA INCLUIDOS
   - 3 perfumes precargados:
     * Lancôme Trésor - $66.500
     * Chanel N°5 - $89.000
     * Set Dolce & Gabbana - $125.000
   - Puedes editarlos o eliminarlos desde Productos

6. FUNCIONES PRINCIPALES
   - VENTA RÁPIDA: Toca producto, agrega al carrito, elige pago
   - PRODUCTOS: Agrega con foto, edita, elimina
   - BODEGA: Controla stock, registra mermas
   - VENTAS: Historial completo con filtros
   - CLIENTES: Registro con teléfono y dirección
   - DEUDAS: Fiado con cuotas automáticas (1-12 meses)
   - CATÁLOGO: Vista previa para compartir
   - CONFIGURACIÓN: Cambia logo, nombre, backup

7. SIN INTERNET DESPUÉS DE CARGAR
   - Una vez abierto, funciona 100% offline
   - Los datos se guardan en el celular
   - NO necesitas WiFi para vender

8. SIN CLAVE / SIN LOGIN
   - Abre directo, sin contraseñas
   - Para uso personal tuyo

9. COLORES
   - Turquesa y azul verdoso (colibrí)
   - Diseño joven, ágil, elegante

10. SOPORTE
    - Si algo no funciona: recarga la página
    - Si se pierden datos: importa tu backup
    - Para borrar todo: Configuración → Borrar datos
